# Oi Skills

**Oi Skills** is an AI design skill suite from Alibaba Cloud Design. Install and invoke it via `/oi-*` in Qoder, Claude Code, Cursor, terminals, and agents. Maintained by Alibaba Cloud designers, with curated open-source design skills (upstream copyright remains with original authors)—covering page styles, motion, UX insights, multimodal generation, and design widgets.

## Install

**Users**

```bash
npx skills add organic-design-ai/oi-skills --all -y -g
```

**Local dev & test**

```bash
cd ..
npx skills add ./oi-skills --all -y -g
```
